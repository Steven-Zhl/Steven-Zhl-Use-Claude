## 1. 主布局（顶部导航 + 侧边栏）

**适用场景**：管理后台、复杂系统

```
┌─────────────────────────────────────────────────────────────────┐
│  Logo  首页  模块1  模块2               搜索框    用户头像 ▼     │  ← el-header (fixed)
├──────────┬──────────────────────────────────────────────────────┤
│          │  面包屑 > 一级 > 二级                                │
│   侧边   │  ┌────────┐  ┌────────┐           [+ 新建]  [导出]  │  ← 操作栏
│   菜单   │  │ 筛选1 │  │ 筛选2 │  [搜索]  [重置]              │
│          │  └────────┘  └────────┘                              │
│  📁 模块1│  ┌────────────────────────────────────────────────┐  │
│  📁 模块2│  │                                                 │  │
│  📁 模块3│  │          主内容区                                │  │
│  ─────── │  │          (el-table / el-card 列表)               │  │
│  📁 模块4│  │                                                 │  │
│          │  └────────────────────────────────────────────────┘  │
│          │  ◀  1  2  3  ...  10  ▶   共 100 条                  │  ← el-pagination
└──────────┴──────────────────────────────────────────────────────┘
```

**响应式适配**：

```
移动端 (<768px):
┌─────────────────────────┐
│ ☰  Logo              ≡ │  ← 汉堡菜单（侧边栏隐藏，点击滑出）
├─────────────────────────┤
│ 面包屑                  │
│ ┌─────────────────┐     │
│ │ 筛选1 (可折叠)   │     │
│ └─────────────────┘     │
│ [搜索]    [+ 新建]      │
├─────────────────────────┤
│ ┌─────────────────────┐ │
│ │  卡片1              │ │  ← 表格转为卡片
│ │  标题: xxx          │ │
│ │  状态: xxx          │ │
│ │  [操作] [详情]      │ │
│ └─────────────────────┘ │
│ ┌─────────────────────┐ │
│ │  卡片2              │ │
│ └─────────────────────┘ │
│    ──── 加载更多 ────   │  ← 分页改为"加载更多"按钮
└─────────────────────────┘
```

## ASCII 绘图技巧

### 常用字符

| 字符 | 用途 |
|------|------|
│ `─` | 水平线 |
│ `│` | 竖线 |
│ `┌` `┐` `└` `┘` | 角落 |
│ `├` `┤` `┬` `┴` `┼` | T型连接 |
│ `□` `○` `●` | 复选框、单选框、状态指示 |
│ `▶` `◀` `▼` `▲` | 箭头/下拉图标 |
│ `[` `]` | 按钮 |
│ `┌─────────┐` | 卡片/弹窗 |

### 绘制步骤

1. **先定外框**：用 `┌─┐│└─┘` 画出整体轮廓
2. **划分区域**：用 `├─┤` 横向分隔，`│` 竖向分隔
3. **填充内容**：从左到右、从上到下填充文字
4. **添加图标**：用简单符号表示按钮、状态、图标
5. **保持对齐**：用等宽字体编辑器（推荐 VS Code + 预览）

### 注意事项

- **不追求美观**：线框图只表达结构关系，不是最终设计稿
- **保持简单**：避免过度细节，重点在布局和层次
- **标注交互**：用箭头或文字说明点击后的行为
- **标注响应式**：如有移动端适配，单独绘制移动端布局
